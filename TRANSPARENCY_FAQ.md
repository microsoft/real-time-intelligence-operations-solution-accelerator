## Does this Solution Accelerator use Generative AI?  

The architecture and resources associated with this solution accelerator do not use Generative AI explicitly. However, the solution accelerator provides guidance for users to create a Microsoft Fabric Data Agent that will provide a chat interface to users about specific data sets. For this solution accelerator, we provided master prompts and other setup instructions for users to test, use, and update. For more information, please refer to [Fabric Data Agent Guide](./docs/FabricDataAgentGuide.md).
